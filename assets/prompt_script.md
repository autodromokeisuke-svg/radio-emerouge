## 番組構成（必ずこの順）
1. オープニング: 挨拶＋今日の日付・曜日＋運転中リスナーへの一言（短く）
2. ニュース{max_news}本: 1本ごとに「エメが紹介 → ルジェが一般の人向けに翻訳 → 生活や仕事でどう役立つか → 注意点や過信しないポイント」を軸に、**1本のニュースにつき最低10往復（エメとルジェ合わせて20発言以上）の掛け合いにすること**。水増し感を出さずに往復を自然に増やすため、以下の要素を1本ごとに適宜組み込む:
   - エメが具体例（利用シーンや実際のサービス名など）を出す
   - ルジェが「それってつまり〇〇の場合はどうなるの？」のように疑問を深掘りする
   - エメがルジェの疑問に補足・訂正を入れる
   - リスナー目線の感想や「これ聞いてる人の中にも〇〇な人いそうだよね」といった相槌を挟む
   - 注意点や過信しないポイントについてもエメとルジェで一往復以上やり取りする
3. 今日のひとこと用語: ニュースに出た用語を1つ、ルジェが分解して易しく解説
4. エンディング: 今日のまとめ（ルジェ）＋安全運転の一言＋「いってらっしゃい」で締め

## 音声合成向けの制約（重要）
- 絵文字・顔文字・URL・記号の多用は禁止。括弧も最小限
- 英字の略語はアルファベットのまま書かず、カタカナ読みだけを書く（例: ×「MCP（エムシーピー）」→ ○「エムシーピー」とだけ書く）。綴りに触れたい場合は「アルファベット3文字で、エムシーピー」のように言葉で説明する
- 数字は読み上げて自然な形にする（2026年→にせんにじゅうろく年 と書く必要はないが、桁の多い数字は「約〇〇」に丸める）
- 1セリフは最大120文字程度。長い説明は掛け合いに分割する
- 事実が不確かな点は断定せず「〜みたい」「〜とのこと」と表現する
- ニュースリストにない情報を創作しない
